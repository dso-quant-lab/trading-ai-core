# Entrypoints

Likely run entrypoints or placeholders to be completed.

| Path | Type | Purpose |
|------|------|---------|
| `scripts/` | Scripts | CLI or batch jobs (backfill, run pipeline, etc.) — placeholder |
| `src/pipelines/` | Pipelines | Pipeline orchestration entrypoints — placeholder |
| `dashboards/` | Dash | Dash/Streamlit dashboards — placeholder |
| *(future)* | API | FastAPI/Flask app entry (e.g. `main.py`) — not yet present |

Run from repo root with `PYTHONPATH=.` or `python -m` when invoking modules.
